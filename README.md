# Vaškovo kolejiště

Statický web připravený pro GitHub Pages.

## Stránky

- `index.html` – úvodní stránka
- `projekt.html` – příběh a vývoj kolejiště
- `lokomotivy.html` – katalog lokomotiv

## Přidání další lokomotivy

1. Vytvořte složku v `assets/images/lokomotivy/`.
2. Nahrajte hlavní fotografii.
3. Otevřete `assets/js/data.js`.
4. Zkopírujte jeden existující blok lokomotivy.
5. Upravte název, typ, texty, cestu k fotografii a technické údaje.

Typ musí být:
- `elektricka`
- `dieselova`

## Výměna ukázkových obrázků

Nahraďte soubory ve složce `assets/images/` vlastními fotografiemi.
Doporučený formát je JPG nebo WebP. Potom upravte příslušnou cestu v HTML nebo v `data.js`.

## GitHub Pages

1. Nahrajte celý obsah této složky do kořene repozitáře.
2. Na GitHubu otevřete `Settings`.
3. Zvolte `Pages`.
4. U `Build and deployment` nastavte `Deploy from a branch`.
5. Vyberte větev `main` a složku `/ (root)`.
6. Uložte nastavení.
